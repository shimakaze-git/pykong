
Pykong
=====

Pykong is a client library of kong.
It's also a command line tool of kong.

Installing
----------

.. code-block:: text

    python setup.py install


Usage
----------------

.. code-block:: text

    pykong api add

    pykong api status

    pykong api get

    pykong api post

    pykong api put

    pykong api delete
